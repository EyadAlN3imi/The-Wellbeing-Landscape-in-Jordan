# The-Wellbeing-Landscape-in-Jordan
# Psychological Resilience, Food Insecurity, and Academic Outcomes Among College Students

## Project Overview

This research project investigates the intricate relationships between socio-economic factors and psychological well-being within a sample of 1000+ college students in Jordan. By employing a mix of advanced statistical modeling and machine learning techniques, the study explores how food insecurity, stress mindsets, and resilience collectively influence mental health and academic success.

The analysis transitions from traditional descriptive statistics to Advanced Inferential Modeling, utilizing Latent Profile Analysis (LPA) to identify distinct student "profiles" and Association Rule Mining to uncover hidden behavioral patterns.

## Project Note
This analysis received a **Full Mark** and was conducted as part of a structured data analysis project using validated psychological measures and statistical modeling.
